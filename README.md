# 綺麗漫画 · KireiManga

KireiManga is a local-first desktop manga reader built on Electron, NestJS and React. It browses MangaDex, manages a local manga library in SQLite, and (in later milestones) overlays OCR + machine translation directly on raw Japanese pages. KireiManga is part of the Shiro suite alongside ShiroAni and Shiranami.
